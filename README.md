# Rohit-Sharma-Performance-Dashboard
This project is an interactive Power BI dashboard created using Rohit Sharma’s cricket performance data from 2019–2025. It provides a clear visual analysis of his batting performance across multiple years, helping users easily understand trends, improvements, and key highlights of his career.


Insights From the Dashboard

=> Performance highs and lows across different years
=> Consistency through batting average and strike rate trends
=> Peak scoring years
=> Milestones like centuries and half-centuries


Project Outcome

This dashboard demonstrates how Power BI can turn player statistics into a storytelling visual report, making insights easy to understand for non-technical users as well.
